Tutorial

Predict captions for your images

Navigate to the predict page by using the navigation bar or just click Predict

You can drag images / click to upload upto 10 images in one request

Once uploaded, a success message appears indicating your request ID that will help track your request and click on the green button to redirect to the processing page

On the processing page, wait for your request to change its status from processing to completed

Once the view results has been enabled, you will be navigated to the results page where you can view the top 3 captions predicted by the model in order of their probability

Use the API for your own applications

Go to the API Endpoint page from the navigation bar or click API Endpoint, create an API key and follow furthur instructions
